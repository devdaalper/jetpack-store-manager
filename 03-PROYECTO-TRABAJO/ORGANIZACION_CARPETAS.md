# Organización de Carpetas (Raíz)

## Carpetas principales

1. `01-WORDPRESS-SUBIR/`
   Contiene únicamente el plugin para WordPress (`jetpack-store-manager/`).
   Esta es la carpeta correcta para generar/subir el ZIP.

2. `02-DESKTOP-BPM-APP/`
   App de escritorio BPM (GUI) para macOS.
   Se abre con `run_gui.command`.

3. `03-PROYECTO-TRABAJO/`
   Archivos de trabajo/desarrollo: `SKILLS`, `AGENTS.md`, `docs`, `tests`, `scripts`, etc.

## Regla simple

- A WordPress solo se sube lo que está en `01-WORDPRESS-SUBIR/jetpack-store-manager/`.
- Nunca subir `02-DESKTOP-BPM-APP/` ni `03-PROYECTO-TRABAJO/`.
